

## Portfolio Website — Senior Embedded Systems Engineer

### Visual Style
- **Dark theme** with a high-tech aesthetic: dark navy/charcoal background, neon accent colors (cyan/green)
- Subtle circuit board patterns or grid lines as background decorations
- Monospace font accents for a "code" feel, clean sans-serif for body text
- Smooth scroll animations as sections come into view

### Pages & Sections (Single Page, English)

1. **Hero Section**
   - Name, title "Senior Embedded Systems Engineer"
   - Animated tagline with key specialties (firmware, RTOS, IoT, etc.)
   - CTA buttons: "View Projects" and "Contact Me"
   - Subtle particle/grid animation in background

2. **About Me**
   - Brief professional summary
   - Profile photo placeholder
   - Key stats (years of experience, projects delivered, etc.)

3. **Technical Skills**
   - Categorized skill cards: Languages (C, C++, Python, Assembly), Platforms (ARM, RISC-V, STM32, ESP32), Tools (JTAG, oscilloscope, logic analyzers), Protocols (SPI, I2C, UART, CAN, MQTT)
   - Visual skill level indicators

4. **Featured Projects**
   - Project cards with thumbnail, title, description, and tech tags
   - Sample projects: IoT sensor network, automotive ECU firmware, industrial control system, wearable device
   - Each card expandable for more details

5. **Experience Timeline**
   - Vertical timeline of career history with company, role, dates, and highlights

6. **Education & Certifications**
   - Degrees and relevant certifications displayed in cards

7. **Contact Section**
   - Contact form (name, email, message)
   - Links to GitHub, LinkedIn, email
   - Location info

8. **Navigation**
   - Sticky top navbar with smooth scroll to sections
   - Mobile hamburger menu
